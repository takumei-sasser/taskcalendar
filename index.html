仕様メモ

  • 月曜日始まりのカレンダーです。
  • position でタスクの表示段数（縦位置）を指定できます。未指定時は 1 になります。
  • 同じ日に position が重複した場合、JSONの出現順に下にずれて表示されます。
  • 期間タスクはセル間で隙間なく連結して表示されます。
  • category と description をサポートしています。
